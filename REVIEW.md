# Review — *Learn Italian the 80/20 Way*

**Originally reviewed:** 2026-06-11 (bare docx) · **Re-evaluated:** 2026-06-19 (after Foundations, missions, study plan, Anki decks, and the Deconstruction rebuild) · **Goal:** near-zero → real-world capability fast, with a concrete questura *permesso di soggiorno* appointment as the live stake.

---

## Verdict (re-evaluated)

The original review called this "a good vocabulary workbook with one structural flaw" — the method was silent reading/writing while the goal was *speaking*. **That flaw is now fixed.** The course has grown a Foundations layer (the Deconstruction step done properly), an explicit three-mode study plan that builds **listening and speaking** alongside writing, spaced-repetition Anki decks, and two concrete real-world missions. It is no longer a workbook; it's a complete fast-track.

**What flipped it:** the [study plan](lessons/study-plan.md) splits each day into INPUT / LISTEN / PRODUCE, so reading-and-writing became the *encoding* layer it should always have been — with audio input and spoken output bolted on either side. The questura mission also correctly reframes the goal as ~80% **listening**, which is the realistic skill for that appointment.

Remaining work is now *polish*, not structural — see "Still open" below.

## Still open (priority order — all polish, not structural)

1. **Make the stake *bite*.** A booked italki lesson is a real stake; a *penalty* is stronger. Add the classic Ferriss mechanism: pre-pay the lesson, or give a friend €100 to donate to a cause you hate if you no-show the questura prep.
2. **Encoding is thin.** Add a lightweight mnemonic habit (vivid image per hard word, or keyword method) so retention isn't relying on repetition alone.
3. **Audio is outsourced.** Listening depends on the learner actually doing Language Transfer/FSI — there's no in-course audio. That's the right 80/20 call, but the plan's success hinges on it; consider a recorded version of the questura officer-command set (§4) since that one is mission-critical and hard to source externally.

The first review's headline risk — *"you can finish this whole plan and still freeze in a real conversation"* — no longer holds. The course now trains input, output, and retention, front-loads the deconstructed grammar, and points at a real spoken stake. With the remaining polish items above (a biting stake, lightweight mnemonic encoding, and optionally recording the questura officer commands), this is a genuinely complete fast track to functional Italian for the questura goal.
