# 🇮🇹 Learn Italian the 80/20 Way

A fast, practical Italian course for adults starting from near-zero — built on the **80/20 principle** (learn the ~20% of the language that does 80% of the work) and **Tim Ferriss' DiSSS framework** (Deconstruction, Selection, Sequencing, Stakes).

The goal isn't fluency for its own sake. It's **real-world capability, fast** — including two concrete missions: completing a *permesso di soggiorno* biometrics appointment at the questura, and holding a casual tennis match in Italian.

## How to use it

1. Read the **[Study Plan](lessons/study-plan.md)** — it tells you exactly what to do in each daily session.
2. Do **[Foundations](lessons/00-foundations.md)** before Day 01.
3. Work one day-lesson per weekday morning; listen at midday; drill Anki + write in the evening.
4. Keep three notebooks beside you: 📖 Dictionary · ✍️ Daily Writing · ❌ Mistakes.
