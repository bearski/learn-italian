# Day 06 — First Sentences
**Week 1 · Focus: Build**

Using **only words you've written so far**, write **3 sentences about today**. No new vocabulary, no dictionary.

Starters: *Io vado... / Voglio... / Non...*

This is the whole game — recombining what you already own. If you don't know a word, describe around it.

---
**Did I...** ☐ write by hand ☐ say each word aloud ☐ log mistakes ☐ make ≥1 original sentence
