# Day 10 — Adjectives
**Week 2 · Focus: Describe**

Write all **adjective pairs**. Write a sentence using each pair, making the adjective **agree** with its noun.

| Italian | English | Opposite |
|---|---|---|
| buono | good | cattivo / brutto (bad) |
| grande | big / great | piccolo (small) |
| nuovo | new | vecchio (old) |
| bello | beautiful | brutto (ugly) |
| giusto | right / correct | sbagliato (wrong) |
| facile | easy | difficile (hard) |
| primo | first | ultimo (last) |
| stesso | same | diverso (different) |
| altro | other / another | — |
| tutto | all / everything | niente (nothing) |
| qualche | some | nessuno (none) |
| poco | little | tanto / molto (a lot) |

**Agreement (see Foundations 0.2):** the adjective copies the noun's gender & number —
*un libro nuovo → due libri nuovi*, *una casa nuova → due case nuove*.
