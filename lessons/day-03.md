# Day 03 — Top 10 Verbs
**Week 1 · Focus: Core Verbs**

Write the **top 10 verbs** with their *io / tu / lui-lei* forms. Say each aloud; add a sentence per verb.

| Infinitive | English | io / tu / lui-lei |
|---|---|---|
| essere | to be | sono / sei / è |
| avere | to have | ho / hai / ha |
| fare | to do / make | faccio / fai / fa |
| andare | to go | vado / vai / va |
| volere | to want | voglio / vuoi / vuole |
| potere | can / be able | posso / puoi / può |
| dovere | must / have to | devo / devi / deve |
| sapere | to know (facts) | so / sai / sa |
| dire | to say / tell | dico / dici / dice |
| vedere | to see | vedo / vedi / vede |

> 🔎 **Foundations hook:** see **0.3 Verb Patterns** — these are the irregular heavy-hitters, but most verbs follow the regular -are / -ere / -ire endings. Learn the pattern, not 28 isolated words.
