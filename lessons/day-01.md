# Day 01 — Function Words
**Week 1 · Focus: Copy & Say**

Write **all function words**, saying each aloud. For every word, write a short sentence on the same line.

| Italian | English | Example |
|---|---|---|
| e / ed | and | pane e acqua |
| ma | but | voglio ma non posso |
| o | or | caffè o tè? |
| di | of / from | una tazza di caffè |
| a | to / at | vado a Roma |
| in | in | vivo in Italia |
| con | with | vengo con te |
| per | for | è per te |
| su | on / about | un libro su Roma |
| da | from / since | vengo da Milano |
| che | that / which / who | la persona che conosco |
| sì | yes | Sì, grazie! |
| si | oneself / one | si chiama Marco |
| no / non | no / not | non capisco |
| anche | also / too | anch'io! |
| più | more | voglio più caffè |
| molto | very / a lot | molto bene! |
| solo | only / just | solo un momento |
| già | already | ho già mangiato |
| ancora | still / again | ancora una volta |
| sempre | always | arrivi sempre tardi |
| mai | never | non vado mai lì |
| qui / qua | here | vieni qui! |
| lì / là | there | è là |
| come | how / like | come stai? |
| quando | when | quando arrivi? |
| dove | where | dove sei? |
| perché | why / because | perché sei triste? |
| se | if | se vuoi, vieni |

**Daily routine (15 min):**
1. **Copy & Say (5 min)** — write each word by hand, say it aloud, and add a short sentence on the same line. Never copy a word in isolation. *voglio → Voglio un caffè.*
2. **Cover & Recall (5 min)** — cover the Italian, write it from memory. Every miss goes in your **Mistakes** notebook, circled.
3. **One Sentence (5 min)** — take 2–3 words and make one sentence. Simple wins. *Voglio un cane grande.*

> 🔎 **Foundations hook:** before you start, read **0.1 Pronunciation** — get *c/g*, *gli*, *gn*, *sc* and double consonants right from word one (*perché*, *già*, *anche*).
