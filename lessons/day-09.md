# Day 09 — Time & Objects
**Week 2 · Focus: Nouns + Time**

Write all **Time** words and **Objects**. Then write **3 sentences**, each using at least one time word.

*Domani vado... / Oggi ho... / Ieri ho mangiato...*

**Time:** oggi (today), ieri (yesterday), domani (tomorrow), ora/adesso (now),
mattina (morning), sera (evening), notte (night), settimana (week), mese (month)

**Objects & Everyday:** cosa (thing), acqua (water), cibo/mangiare (food), soldi (money),
macchina (car), telefono (phone), lavoro (work/job), numero (number), parola (word)

> 🔎 **Foundations hook:** "Ieri ho..." needs the **past tense (passato prossimo)** — *avere/essere + participle*. See **0.3**: *Ho mangiato. / Sono andato.*
