# Day 11 — Survival Phrases
**Week 2 · Focus: Memorise**

Write all **14 survival phrases**. Read each aloud. For each one, picture the exact moment in Italy you'd need it.

| Italian | English |
|---|---|
| Non capisco. | I don't understand. |
| Puoi ripetere? | Can you repeat? |
| Come si dice...? | How do you say...? |
| Non lo so. | I don't know. |
| Dove si trova...? | Where is...? |
| Quanto costa? | How much does it cost? |
| Mi piace. | I like it. |
| Ho bisogno di... | I need... |
| Mi scusi. | Excuse me. |
| Grazie mille. | Thank you very much. |
| Prego. | You're welcome. |
| Mi dispiace. | I'm sorry. |
| Va bene. | That's fine / OK. |
| Hai ragione. | You're right. |

These save you in real conversations — drill them until they're automatic.
