# Day 05 — The Remaining Verbs
**Week 1 · Focus: More Verbs**

Write the **remaining verbs**. Say each aloud; add a sentence per verb.

| Infinitive | English | io / tu / lui-lei |
|---|---|---|
| venire | to come | vengo / vieni / viene |
| conoscere | to know (people) | conosco / conosci / conosce |
| dare | to give | do / dai / dà |
| stare | to stay / be | sto / stai / sta |
| prendere | to take | prendo / prendi / prende |
| mettere | to put | metto / metti / mette |
| parlare | to speak | parlo / parli / parla |
| capire | to understand | capisco / capisci / capisce |
| pensare | to think | penso / pensi / pensa |
| trovare | to find | trovo / trovi / trova |
| tornare | to return | torno / torni / torna |
| mangiare | to eat | mangio / mangi / mangia |
| bere | to drink | bevo / bevi / beve |
| uscire | to go out | esco / esci / esce |
| lavorare | to work | lavoro / lavori / lavora |
| chiamare | to call | chiamo / chiami / chiama |
| vivere | to live | vivo / vivi / vive |
| usare | to use | uso / usi / usa |

> 🔎 **Foundations hook:** *capire* shows the **-isc-** pattern (capisco, capisci, capisce…). It also covers *finire*, *preferire*. See **0.3**.
