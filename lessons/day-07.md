# Day 07 — Review Day
**Week 1 · Focus: Consolidate**

**Sunday-style review (30 min):**
1. Open your **Mistakes** notebook. Rewrite each error **3 times**, correctly.
2. Write **one new sentence** using each mistake word.
3. Write **5 sentences** using any words from the week.
4. In your Daily Writing, put a tally next to every word you used more than once — those are *your* high-frequency words. Study them hardest.
