# 🎾 Mission Lesson — Conversational on the Tennis Court

**Goal:** play a casual match in Italian — call the score, handle the points, and chat at the changeovers and after. **A tennis match is a closed, repetitive domain** (~50 phrases), most of which repeat every few points. You do **not** need to be generally conversational — you need the *court script*.

> **The honest truth:** scoring is the spine — you say it on every point, so it drills itself. Add ~15 on-court lines and ~8 small-talk lines and you can carry a whole match. Rehearse out loud *while you play* — the physical action pins the word, exactly like the questura listening drill.

---

## 1. Scoring — the spine (say this every point)

Italian tennis uses the same 15 / 30 / 40 system, spoken aloud.

| Italian | Meaning |
|---|---|
| zero | love / 0 |
| quindici, trenta, quaranta | 15, 30, 40 |
| **quindici pari** / **trenta pari** | 15-all / 30-all |
| **parità** *(or quaranta pari)* | deuce (40-40) |
| **vantaggio** (mio / tuo) | advantage (me / you) |
| il game / il gioco | game |
| il set | set |
| il tie-break | tiebreak |
| **Quanto siamo?** / **Come stiamo?** | What's the score? |

- Server's score first: **"Trenta a quindici"** = 30-15.
- **"Cambiamo campo."** = Let's change ends.
- **"Set point."** / **"Match point."** — used as-is, English loanwords are fine.

---

## 2. On-court phrases — the 15 that repeat

| Italian | Meaning |
|---|---|
| **Giochiamo?** / **Facciamo un set?** | Shall we play? / Play a set? |
| **Riscaldiamoci.** / **Facciamo un po' di palleggio.** | Let's warm up / knock up. |
| **Tocca a te servire.** / **Servi tu.** | Your serve. |
| **Chi serve?** | Who's serving? |
| **Pronti? Via!** | Ready? Go! |
| **Dentro!** / **Fuori!** | In! / Out! |
| **Era buona.** / **Era fuori.** | It was in / out. |
| **Rete!** / **Net!** | Let (off the net cord). |
| **Bel colpo!** / **Bel punto!** | Nice shot / point! |
| **Che palla!** | What a shot! |
| **Doppio fallo.** | Double fault. |
| **La mia!** / **Tua!** | Mine! / Yours! *(in doubles)* |
| **Un'altra?** | Another one? |
| **Bella partita!** | Good match! |
| **Rivincita?** | Rematch? |

> 🔎 When a call is disputed, the friendly default is **"Ripetiamo il punto."** — let's replay the point.

---

## 3. Small talk — changeovers & after the match

| Italian | Meaning |
|---|---|
| **Giochi spesso?** | Do you play often? |
| **Da quanto tempo giochi?** | How long have you played? |
| **Hai un bel rovescio.** | You've got a nice backhand. |
| **Sono un po' arrugginito.** *(m)* / **arrugginita** *(f)* | I'm a bit rusty. |
| **Oggi non sono in forma.** | I'm not on form today. |
| **Che caldo oggi!** | So hot today! |
| **Mi hai battuto, complimenti.** | You beat me, well done. |
| **Beviamo qualcosa?** | Shall we grab a drink? |

## 5. Verbs you'll actually use

- **giocare** (a tennis) — to play · *Gioco a tennis il sabato.*
- **servire** — to serve · *Servi tu.*
- **colpire / tirare** — to hit · *Hai colpito bene.*
- **vincere / perdere** — to win / lose · *Ho perso il primo set.*
- **battere** — to beat · *Mi hai battuto.*

> 🔎 **Foundations hook:** "Ho perso / Hai colpito" is the **passato prossimo** (avere + participle). See **[0.3](00-foundations.md)**.

---

## 6. A whole match in Italian (read it through)

> — **Giochiamo un set?** — Sì! **Riscaldiamoci** prima.
> — **Servi tu.** … **Pronti? Via!**
> — **Quindici a zero.** … **Bel punto!** … **Trenta pari.**
> — *(net cord)* **Rete! Ripetiamo il punto.**
> — **Vantaggio mio.** … **Game.** **Cambiamo campo.**
> — … **Match point.** … **Fuori!** … **Bella partita! Mi hai battuto.**
> — **Rivincita la prossima volta? Beviamo qualcosa?**

---

## 7. 2-week drill plan

Layer this on top of the 14-day foundation — it's small.

- **Days 1–3:** memorise **Section 1 (scoring)** cold. Say a fake score aloud for every point of a real or imagined game.
- **Days 4–7:** add **Section 2** — call your own shots out loud while practising against a wall or hitting partner.
- **Days 8–10:** add **Section 3** small talk; run the Section 6 script out loud, both sides.
- **Days 11–14:** play a real (or imagined) match narrating **everything** in Italian. **Stakes:** tell a hitting partner you'll only call the score in Italian for one full set.

---

## 8. Emergency minimum (4 phrases)

1. **Quanto siamo?** *(reset the score anytime.)*
2. **Dentro! / Fuori!** *(make calls.)*
3. **Bel punto! / Bel colpo!** *(stay friendly.)*
4. **Bella partita!** *(close it well.)*

*Buon divertimento — in campo!* 🎾
